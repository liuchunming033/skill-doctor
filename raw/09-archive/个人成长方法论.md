
# 「个人 Slogan」
**跟随实干者成长，用费曼法做内容、做产品，输出观点。**
# 一、输入：follow builders not influencers

## 1. 输入原则

- 不追网红、不追情绪、不追流量话术
- **只跟随实干者、开发者、产品人、创作者（Builders）**
- 所有输入：**少而精、重原理、轻碎片**

## 2. 输入筛选标准

1. 对方**有落地作品**（产品 / 工具 / 内容体系），不是只会讲
2. 内容有**底层逻辑**，不是技巧堆砌
3. 持续在建东西，而不是持续蹭流量

# 二、输出：（产品 + 内容，全部费曼式输出）

### 输出 1：做产品（Build）

- 自己动手搭建、制作、改造工具 / 流程 / 体系
- **输出产品观点**：为什么这么做、解决什么痛点、取舍是什么
- 输出形式：工具、流程、模板、系统、自动化方案

### 输出 2：做内容（Explain）

- 把自己「学到的、做到的、踩过坑的」用大白话讲出来
- **输出认知观点**：方法论、思维、结构、套路
- 输出形式：拆解、复盘、结构分析、思维模型

## 输出铁律

**所有内容，必须带「个人观点」，不搬运、不复读、不碎碎念。** **标题 = 我的结论 / 我的方案 / 我反对的点**

没有观点不发，没有思考不发，没有实践不发。

不用铺垫、不用设问、不用废话。

观众点开就是：**你已经思考完、给答案了**。

# 三、完整闭环

1. **精输入**：follow builders，吸收实干底层逻辑
2. **内化**：自己动手做产品、做实践
3. **费曼输出**：用内容讲清楚、讲通透、讲出自己观点
4. **沉淀产品**：内容沉淀成模板 / 工具 / 体系
5. **反向筛选更高阶输入**
