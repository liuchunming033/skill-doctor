## 核心摘要
腾讯 TEG 云架构平台部网关测试团队在 Agent 测评领域的体系化实践。面对 Agent 非确定性、黑盒化、错误级联放大三大难题，建立了"确定性评分器 + Rubric 评分器 + 人工评分器"三类组合的完整测评框架，覆盖功能正确性、过程质量、效率成本、鲁棒性安全、体验对齐五大维度。在 TPerf 性能平台智能分析 Agent 项目中落地验证，包含 9 类场景 30+ 用例、多模型对比（7 个模型）、基线管理、CI 自动化等完整工程实现。
