## 核心摘要
OpenAI 平台提供了以 Trace（执行轨迹）为核心的 Agent 评估工具链。评估流程从 Trace 调试开始（快速识别工作流级别问题），然后过渡到数据集和评估运行（用于可重复的基准测试）。Trace 评分适用于回答"Agent 选对工具了吗？""交接发生对了吗？"等问题；数据集和 Eval 运行则用于基准对比和大规模评估。
