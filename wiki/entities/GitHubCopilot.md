## 定义

GitHub Copilot，由 Microsoft / GitHub 开发的 AI 编程助手，是当前企业级 AI 编码工具的代表产品之一。

## 关键信息

**研究数据（Microsoft / GitHub 大规模样本）：**

- 任务完成速度提升 **55.8%**
- 任务成功率提升 **78%**

**企业部署案例：**

- [[Accenture]]：PR 周期缩短 75%（9.6天 → 2.4天），配套审查者赋能流程
- 腾讯：90% 工程师使用，50% 新增代码由 AI 辅助生成

**产品定位**：属于 Sync Agent 类工具（实时编程辅助），介于基础补全和完全自主的 Async Agent 之间。
