## 定义

Anthropic 是一家专注于 AI 安全和可靠性的公司，开发了 Claude 系列 AI 模型。公司于 2024 年 10 月 16 日发布官方文档《Equipping agents for the real world with Agent Skills》，提出 Agent Skills 机制解决 Agent 在真实世界任务中的瓶颈。

## 关键信息

### 公司定位

- **核心理念**：AI 安全优先，追求可靠、可解释的 AI 系统
- **核心产品**：Claude 系列 AI 模型（Claude 3.5 Sonnet、Claude 3 Opus 等）
- **技术贡献**：提出 Agent Skills 概念，推动 Agent 从"会聊天"转向"能做事"

### Agent Skills 文档贡献

2024 年 10 月 16 日发布的文档《Equipping agents for the real world with Agent Skills》指出：

- Agent 面临两大能力缺口：过程知识缺失和组织背景缺失
- 提出 Skill 作为解决方案：由指令、脚本、资源三要素组成
- 强调动态加载机制：Skill 可被动态检索、挂载和使用
- 区分 Skill vs Tool：Skill 是调用 Tool 的 SOP，封装过程知识

### 相关产品

- **Claude**：Anthropic 开发的大语言模型，具备强大对话和推理能力
- **Claude Code**：Anthropic 推出的 AI 编程助手，内置 Skills 机制
